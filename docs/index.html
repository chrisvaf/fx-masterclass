Feature Experimentation Demos

To get started, enter valid SDK Key by clicking the "Details" button in the top nav.

Once you have entered a valid Feature Experimentation SDK Key you can explore multiple demo scenarios leveraging the power of Optimizely's Feature Experimentation platform. This site is built using simple technologies, outlined below. Optimizely's JavaScript (Browser) SDK is being used to execute feature flags and experiments.

A utility-first CSS framework for rapidly building custom designs.
